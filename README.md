# Bus Simulator Game

## Project Overview
This project is a Unity-based bus simulator game featuring bus physics, a route/map system, traffic simulation, and a passenger system.

## Features
- **Bus Physics**: Realistic bus dynamics that simulate weight, acceleration, and braking.
- **Route/Map System**: Manage multiple routes and customizable maps.
- **Traffic Simulation**: AI-controlled vehicles that follow traffic laws and interact with the player.
- **Passenger System**: Ability to pick up and drop off passengers, along with ticket management.
